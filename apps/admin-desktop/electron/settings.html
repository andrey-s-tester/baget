Подключение

Интерфейс админки встроен в приложение. Укажите URL вашего API (Nest), к которому будет обращаться админка.

Адрес, где у вас открыта админка в браузере (режим без встроенного UI).

В режиме с встроенной админкой обязателен. Для удалённого URL админки — базовый адрес бэкенда (если не задан, берётся из URL админки).

Если поле пустое, программа сама вычислит адрес от URL API или (в режиме внешней админки) от URL админки. На сервере API должны быть заданы DESKTOP_ADMIN_LATEST_VERSION и DESKTOP_ADMIN_DOWNLOAD_URL.

Обычно путь: …/api/system/desktop-admin-update — в ответе JSON с полями latest и downloadUrl. Вставить из буфера в поле: ПКМ → Вставить или меню Правка → Вставить (Ctrl+V).
